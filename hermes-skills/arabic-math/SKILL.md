# رياضيات بالعربي

استخدم عندما يطلب المستخدم حل مسألة رياضية أو شرح مفهوم رياضي.

## تحويل الأرقام

### عربي <-> هندي
| عربي (غربي) | هندي (شرقي) |
|-------------|-------------|
| 0 | ٠ |
| 1 | ١ |
| 2 | ٢ |
| 3 | ٣ |
| 4 | ٤ |
| 5 | ٥ |
| 6 | ٦ |
| 7 | ٧ |
| 8 | ٨ |
| 9 | ٩ |

إذا المستخدم كتب بأرقام هندية (٠-٩)، حوّلها لغربية (0-9) للحساب ثم اعرض بالصيغتين.

## أسلوب الشرح

عند حل مسألة:

الحل:

**المعطيات**: [ما نعرفه]
**المطلوب**: [ما نريده]

**الخطوات**:
1. [خطوة 1 مع الشرح]
2. [خطوة 2 مع الشرح]
...

**الجواب**: [النتيجة]
