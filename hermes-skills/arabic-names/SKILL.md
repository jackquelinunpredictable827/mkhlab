# أسماء عربية

استخدم عندما يسأل المستخدم عن معنى اسم أو يريد اقتراح أسماء.

## متى تستخدم
- المستخدم يسأل "وش معنى اسم [اسم]؟"
- يريد اقتراحات أسماء لمولود
- يريد أسماء بمعنى معين

## البحث عن معنى اسم

عند السؤال عن معنى اسم:

اسم: [الاسم]

- **المعنى**: [المعنى]
- **الأصل**: عربي / فارسي / تركي / أمازيغي
- **النوع**: مذكر / مؤنث / مشترك
- **مشاهير بهذا الاسم**: [أمثلة]
- **حكم التسمية**: جائز / مكروه / محرم (مع السبب)

## اقتراح أسماء

عند طلب اقتراحات:
1. اسأل: ولد أم بنت؟
2. اسأل: تفضيل معين؟ (قرآني، كلاسيكي، حديث، قصير)
3. اقترح 10 أسماء مع معانيها:

### أسماء قرآنية مقترحة (أولاد)
آدم، إبراهيم، يوسف، يونس، إلياس، إدريس، لقمان، ذكريا

### أسماء قرآنية مقترحة (بنات)
مريم، آسية، هاجر، سارة، خديجة

### أسماء معانيها جميلة
| الاسم | المعنى |
|-------|--------|
| فارس | الفروسية والشجاعة |
| غيث | المطر |
| ريان | باب الجنة للصائمين |
| نايف | العالي المرتفع |
| لين | النعومة والرقة |
| رزان | الوقار والرزانة |
| تالا | النخلة الصغيرة |
| ميار | جالبة الخير |

## أحكام التسمية

### ممنوع
- التسمية بأسماء الله الخاصة (الله، الرحمن)
- أسماء تعبّد لغير الله (عبدالنبي، عبدالكعبة)
- أسماء فيها معنى سيء أو تشاؤم

### مكروه
- أسماء فيها تزكية مبالغة (بَرّة)
- أسماء ملائكة (جبريل — خلاف بين العلماء)

### مستحب
- عبدالله، عبدالرحمن (أحب الأسماء إلى الله)
- أسماء الأنبياء
- أسماء ذات معنى حسن

## القواعد
- لا تفتِ بالحرام/الحلال — قل "جمهور العلماء يرى..."
- احترم ثقافة المستخدم — الأسماء تختلف بين المناطق
- إذا الاسم غير عربي (تركي، فارسي)، وضّح الأصل
