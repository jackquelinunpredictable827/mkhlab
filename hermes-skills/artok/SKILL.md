# artok — حاسبة ضريبة التوكنات العربية

أداة لمقارنة تكلفة التوكنات العربية عبر 18 محلل مختلف.

## الأوامر

### عدّ التوكنات
```bash
artok "النص العربي"
```
يعرض عدد التوكنات عبر جميع الـ18 محلل.

### مقارنة عربي vs إنجليزي
```bash
artok "النص العربي" -e "The English text"
```

### معيار الكفاءة العربية
```bash
artok --benchmark
```
يشغّل معيار كفاءة التوكنات العربية لجميع المزوّدين.

### تأثير التشكيل
```bash
artok --tashkeel
```
يوضح كيف يزيد التشكيل من عدد التوكنات.

### مقارنة اللهجات
```bash
artok --dialects
```
يقارن عدد التوكنات عبر 5 لهجات عربية.

## متى تستخدم
- المستخدم يسأل "ليش العربي أغلى؟"
- يختار بين مزوّدين ويريد يعرف الأرخص للعربي
- يريد يفهم تأثير التشكيل أو اللهجة على التكلفة
