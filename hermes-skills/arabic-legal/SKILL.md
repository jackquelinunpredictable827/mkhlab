# مصطلحات قانونية عربية

استخدم عندما يسأل المستخدم عن قانون أو عقد.

## تنبيه
مخلب لا يقدم استشارات قانونية. المعلومات تثقيفية فقط.
دائماً قل: "استشر محامي متخصص."

## مصطلحات أساسية

| الإنجليزي | العربي | الشرح |
|-----------|--------|-------|
| Contract | عقد | اتفاق ملزم بين طرفين |
| Clause | بند | فقرة في العقد |
| Liability | مسؤولية | التزام قانوني |
| Warranty | ضمان | كفالة جودة أو أداء |
| Indemnity | تعويض | دفع مقابل ضرر |
| Force Majeure | قوة قاهرة | ظرف خارج السيطرة |
| Arbitration | تحكيم | حل نزاع خارج المحكمة |
| Jurisdiction | اختصاص قضائي | صلاحية المحكمة |
| Power of Attorney | وكالة | تفويض شخص بالتصرف |
| Notarization | توثيق | تصديق رسمي |
| Plaintiff | مدّعي | من يرفع الدعوى |
| Defendant | مدّعى عليه | من تُرفع عليه الدعوى |
| Appeal | استئناف | طعن في حكم |
| Verdict | حكم | قرار المحكمة |
| Settlement | تسوية | حل ودّي |
| Intellectual Property | ملكية فكرية | حقوق الابتكار |
| NDA | اتفاقية سرية | عدم إفشاء معلومات |
| Terms of Service | شروط الخدمة | قواعد الاستخدام |
| Privacy Policy | سياسة الخصوصية | كيفية التعامل مع البيانات |
| Waiver | تنازل | إسقاط حق |

## أنواع العقود الشائعة
| العقد | الاستخدام |
|-------|-----------|
| عقد عمل | توظيف |
| عقد إيجار | تأجير عقار |
| عقد بيع | شراء/بيع |
| عقد شراكة | تأسيس شراكة |
| عقد استشاري | خدمات استشارية |
| عقد مقاولة | أعمال بناء/تنفيذ |
| اتفاقية عدم إفشاء | حماية معلومات سرية |
| اتفاقية عدم منافسة | منع المنافسة بعد ترك العمل |

## القواعد
- لا تصيغ عقود قانونية — ساعد في الفهم فقط
- دائماً: "استشر محامي"
- القوانين تختلف بين الدول العربية — اسأل عن الدولة
- للعقود الإسلامية (مرابحة، إجارة): وجّه لـ islamic-finance skill
