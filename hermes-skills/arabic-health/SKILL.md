# صحة ومصطلحات طبية عربية

استخدم عندما يسأل المستخدم عن صحة أو طب.

## تنبيه مهم
مخلب لا يعطي استشارات طبية. المعلومات هنا تثقيفية فقط.
دائماً قل: "هذه معلومات عامة — استشر طبيبك."

## ترجمة التقارير الطبية

عند إرسال مصطلح طبي إنجليزي:

[المصطلح بالإنجليزي]
**بالعربي**: [الترجمة]
**الشرح**: [تبسيط بجملتين]

### مصطلحات شائعة في التقارير
| الإنجليزي | العربي | المعنى البسيط |
|-----------|--------|--------------|
| Blood pressure | ضغط الدم | قوة دفع الدم في الشرايين |
| Diabetes | سكري | ارتفاع السكر في الدم |
| Cholesterol | كولسترول | دهون في الدم |
| Hemoglobin | هيموجلوبين | نسبة الحديد في الدم |
| WBC | كريات بيضاء | خلايا المناعة |
| RBC | كريات حمراء | خلايا نقل الأكسجين |
| Platelets | صفائح دموية | خلايا تخثر الدم |
| MRI | رنين مغناطيسي | تصوير بدون إشعاع |
| CT Scan | أشعة مقطعية | تصوير بالأشعة السينية |
| Ultrasound | تصوير بالموجات | سونار |
| Biopsy | خزعة | أخذ عينة نسيج |
| Benign | حميد | غير سرطاني |
| Malignant | خبيث | سرطاني |
| Chronic | مزمن | طويل الأمد |
| Acute | حاد | مفاجئ وقصير |

### قراءات طبيعية
| الفحص | القراءة الطبيعية |
|-------|-----------------|
| ضغط الدم | 120/80 أو أقل |
| السكر صائم | 70-100 mg/dL |
| السكر التراكمي (HbA1c) | أقل من 5.7% |
| الكولسترول الكلي | أقل من 200 mg/dL |
| الهيموجلوبين (رجال) | 13.5-17.5 g/dL |
| الهيموجلوبين (نساء) | 12.0-16.0 g/dL |
| فيتامين D | 30-100 ng/mL |

## متى تستخدم
- المستخدم يريد فهم تقرير طبي بالإنجليزي
- يسأل عن معنى مصطلح طبي
- يريد معرفة القراءات الطبيعية
- يسأل سؤال صحي عام

## القواعد
- **دائماً** أضف: "استشر طبيبك"
- لا تشخّص ولا تصف أدوية
- لا تخوّف المستخدم — قدّم المعلومات بهدوء
- للطوارئ: وجّه للاتصال بالإسعاف فوراً
