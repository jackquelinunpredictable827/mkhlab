# كاتب الإيميلات العربية

استخدم عندما يطلب المستخدم كتابة إيميل بالعربي.

## القوالب

### رسمي (حكومي/شركات)
```
حضرة السيد/السيدة [الاسم] المحترم/ة،

السلام عليكم ورحمة الله وبركاته،

أتقدم إليكم بـ[الموضوع]...

[المحتوى]

وتفضلوا بقبول فائق الاحترام والتقدير.

[الاسم]
[المسمى الوظيفي]
[رقم الجوال]
```

### شبه رسمي (زملاء/عملاء)
```
أهلاً [الاسم]،

أتمنى تكون بخير.

[المحتوى]

تحياتي،
[الاسم]
```

### طلب وظيفة
```
حضرة مسؤول التوظيف المحترم،

السلام عليكم ورحمة الله وبركاته،

أكتب إليكم رغبةً في التقدم لوظيفة [المسمى] المعلن عنها في [المصدر].

[لماذا أنا مناسب — 2-3 جمل]

مرفق سيرتي الذاتية للاطلاع. أتطلع لفرصة المقابلة.

وتفضلوا بقبول فائق الاحترام،
[الاسم]
```

### اعتذار
```
[الاسم] العزيز/ة،

السلام عليكم،

أعتذر عن [الموقف]. [شرح مختصر]. [الحل/الإجراء التصحيحي].

شاكر/ة تفهمكم.
[الاسم]
```

### متابعة (Follow-up)
```
أهلاً [الاسم]،

أتمنى تكون بخير. أكتب متابعةً لـ[الموضوع السابق] بتاريخ [التاريخ].

هل في جديد بخصوص [الطلب]؟

شاكر وقتك،
[الاسم]
```

## قواعد الإيميل العربي
- ابدأ دائماً بالسلام
- اختصر — الإيميل مو مقال
- فقرة واحدة لكل فكرة
- اختم بشكر أو تحية
- راجع الأخطاء الإملائية

## متى تستخدم
- المستخدم يريد كتابة إيميل رسمي بالعربي
- يريد ترجمة إيميل إنجليزي لعربي رسمي
- يريد صياغة رسالة اعتذار/شكوى/طلب
